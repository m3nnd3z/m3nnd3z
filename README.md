## 🧠 Sobre mí

Desarrollador de software especializado en **backend con PHP y Symfony**, con experiencia real en proyectos en producción, incluyendo el desarrollo de un **CRM** con lógica de negocio compleja. Trabajo con **arquitectura por capas / hexagonal** y aplico principios **SOLID** para escribir código mantenible y escalable.

En el frontend combino **React, TypeScript y Next.js**, conectando aplicaciones completas end-to-end. También tengo experiencia con **sistemas asíncronos (RabbitMQ)**, **integraciones externas** (firma electrónica con DocuSeal) y **despliegue en producción** (Docker, Vercel, Render, Scaleway).

🎓 Técnico en Desarrollo de Aplicaciones Multiplataforma (DAM)
🚀 Actualmente en **Altarius ETI**, manteniendo y evolucionando aplicaciones como *Altariuseti.com* y *Kalima.es*
